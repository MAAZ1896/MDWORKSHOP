Introduction

Automated teller machine is a computer controlled money vending machine that allows customers to withdraw cash, generate mini statements, deposit cash, and instantly generate debit card pins.

Prerequisite

  1. The customer must maintain an active account in the bank at any branch across India.
  2. Make sure the mobile number is update with the bank.
  3. The debit card must contain customers name and a valid signature on the back of the card.

How to withdraw cash from the ATM

The ATM screen displays commercial advertisements like home loans, vehicle loans, wait for the screen to display Insert your card in the right bottom corner

Steps to withdraw cash

  1. Insert the debit card into card slot and wait for the system to load your information.
  2. Check the name and account information and select Banking option on the right pane, Features under the Banking options
    • Mini Statement displays transactions for the last ten days.
    • Balance Inquiry displays the available balance.
    • Fast Cash one touch withdrawals, allowing users to select preset amounts.
    • Cash Withdrawal allows standard withdrawals.
    • Deposit allows users to deposit money into their account.
    • Change Pin provides Instant pin generation facility.
  3. Select Cash Withdrawal > Enter Amount.
  4. Manually enter the amount required under Enter Amount and select Correct.
  5. The screen prompts do you want to print the statement, Select Yes to print the statement and select No to proceed without printing.
  6. Enter the registered mobile number, select Confirm and wait for the ATM to dispense the cash.

Note: E Banking options for savings and current account differs with respect to the transaction limits per day.
